# MarketMind-Strategic-Intelligence

## Quick Start (Windows)
Just double-click: START-PROJECT.bat

## Manual Start

### Step 1 - Start Backend
Open terminal in marketmind-backend folder:
```
npm install
node server.js
```
Should show: MongoDB connected + Server running on port 4000

### Step 2 - Start Frontend  
Open NEW terminal in marketmind-frontend folder:
```
npm install
npm run dev
```

### Step 3 - Open Browser
Go to: http://localhost:3000

## API Keys Already Set in .env
- GEMINI_API_KEY (Google AI - Free)
- NEWS_API_KEY (NewsAPI - Free)
- MONGODB_URI (Local MongoDB - Free)

## Requirements
- Node.js installed (nodejs.org)
- MongoDB installed locally (mongodb.com/try/download/community)
- MongoDB must be running as a service

## Features

- Live Competitor Map (OpenStreetMap - Free)
- Revenue Forecasting
- SWOT Generator
- Business Model Canvas
